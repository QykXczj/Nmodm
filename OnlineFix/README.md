# OnlineFix 文件夹说明

## 📁 文件夹内容

这个文件夹包含了《艾尔登法环：夜之王朝》的在线修复文件，用于游戏的破解和在线功能。

### 📋 文件列表

- `OnlineFix.ini` - OnlineFix 主配置文件 ✅
- `dlllist.txt` - DLL 文件列表 ✅
- `OnlineFix64.dll` - 主要的破解 DLL 文件 ⚠️
- `winmm.dll` - Windows 多媒体 DLL 文件 ⚠️
- `gconfig.ini` - 游戏配置文件（运行时生成）⚠️

### ⚠️ 重要说明

**二进制文件未上传到 GitHub**

由于以下原因，二进制 DLL 文件没有上传到 GitHub 仓库：

1. **版权考虑**: 这些文件可能涉及版权问题
2. **安全考虑**: 杀毒软件可能将破解文件标记为恶意软件
3. **GitHub 政策**: GitHub 对破解相关文件有限制
4. **文件大小**: 二进制文件会增加仓库大小

### 🔧 如何获取完整文件

如果你需要完整的 OnlineFix 文件，请：

1. **从可信来源下载**: 从官方或可信的游戏社区获取
2. **本地配置**: 将文件放置在项目的 `OnlineFix/` 目录中
3. **运行应用**: Nmodm 会自动检测和管理这些文件

### 📝 配置说明

#### OnlineFix.ini 配置项：

- `RealAppId=2622380` - 艾尔登法环夜之王朝的 Steam App ID
- `FakeAppId=480` - 用于模拟的 App ID
- `DLC` 部分包含了所有 DLC 的配置

#### 支持的 DLC：

- ELDEN RING NIGHTREIGN CE Content (Retail)
- ELDEN RING NIGHTREIGN - Deluxe Upgrade Pack  
- ELDEN RING NIGHTREIGN Digital Artbook & Mini Soundtrack
- ELDEN RING NIGHTREIGN Bonus Gesture

### 🚀 使用方法

1. 确保 OnlineFix 文件完整
2. 在 Nmodm 中点击"应用破解文件"
3. 应用会自动将文件复制到游戏目录
4. 启动游戏即可享受在线功能
