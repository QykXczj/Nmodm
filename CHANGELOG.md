## [1.0.1] - 2025-01-06

### 新增
- 🔒 **强制最后加载功能** - 在mod配置右键菜单中添加"强制最后加载"选项
  - 支持设置任意mod为强制最后加载
  - 自动配置load_after依赖关系
  - 智能检测当前强制加载状态
  - 支持一键取消强制加载设置

### 修复
- 🐛 **TOML配置格式修复** - 修复ME3配置文件生成的格式问题
  - 修复load_after字段生成错误的嵌套表格格式
  - 现在生成正确的格式：`load_after = [{id = "ModA", optional = true}]`
  - 统一配置预览和实际文件的格式一致性

### 改进
- ⚡ **配置管理优化** - 改进mod依赖管理系统
  - 新增自定义TOML写入方法确保格式正确
  - 优化依赖关系的格式化和显示
  - 增强配置文件的可读性和兼容性

## [1.0.0] - 2025-01-05

### 新增
- 🎮 完整的游戏管理功能
- 🔧 ME3 工具集成和自动下载
- 🎯 智能 Mod 管理系统
- 🌐 多镜像下载支持
- 💬 Mod 注释和备注功能
- 🚀 快速启动界面
- 📦 双打包方案支持 (PyInstaller + Nuitka)
- 🎨 现代化无边框界面设计

### 功能特性
- **游戏配置管理**
  - 游戏路径自动检测和验证
  - OnlineFix 在线修复文件管理
  - 配置文件持久化存储

- **ME3 工具集成**
  - GitHub API 版本检测
  - 多镜像下载 (GitHub, Ghproxy 等)
  - 自动解压和安装
  - 下载进度显示和取消功能

- **智能 Mod 管理**
  - 内部 Mod 自动扫描
  - 外部 Mod 手动添加
  - 智能 Mod 类型检测 (文件夹型/DLL型)
  - DLL 重名冲突检测
  - 实时配置文件生成
  - 右键上下文菜单操作

- **用户界面**
  - 基于 PySide6 的现代化设计
  - 无边框窗口和自定义标题栏
  - 可拖拽窗口
  - 侧边栏导航
  - 内联状态显示 (无弹窗设计)

### 技术实现
- **架构设计**
  - 模块化组件设计
  - 信号槽通信机制
  - 配置管理器模式
  - 异步下载处理

- **打包部署**
  - PyInstaller 单文件/目录模式
  - Nuitka 高性能编译

## 版本说明

### 版本号格式
- **主版本号**: 重大功能更新或架构变更
- **次版本号**: 新功能添加或重要改进
- **修订号**: 问题修复和小幅优化

### 发布周期
- **稳定版**: 每月发布，包含完整测试
- **预览版**: 每周发布，包含最新功能
- **热修复**: 按需发布，修复关键问题

### 支持政策
- **当前版本**: 完整支持和更新
- **前一版本**: 安全更新和关键修复
- **更早版本**: 仅提供社区支持
