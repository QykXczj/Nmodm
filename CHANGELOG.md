# 更新日志

## [1.0.0] - 2025-07-05

### 新增功能
- ✨ 基于 PySide6 的现代化无边框 GUI 界面
- 📦 智能 Mod 管理系统，支持内部和外部 Mod
- 🔧 完整的 ME3 配置文件生成和游戏启动功能
- 🌐 多镜像下载支持，解决网络访问问题
- 🚀 快速启动界面，显示当前配置信息
- 🏷️ Mod 备注功能，便于管理和识别
- 🔍 智能 Mod 类型检测（文件夹型、DLL型、混合型）
- 📱 右键上下文菜单，支持单个 Mod 管理
- 💾 外部 Mod 支持，可添加任意路径的 Mod
- 🎨 三种主题模式（深色、浅色、灰色）
- 📋 可视化配置预览和编辑
- 🔄 实时配置同步和状态更新

### 技术特性
- 🛠️ 模块化架构设计，便于维护和扩展
- 🔗 信号-槽模式，实现组件间通信
- 🚫 无模态对话框设计，避免弹窗干扰
- 📄 JSON/TOML 配置文件管理
- 🧵 多线程下载，不阻塞界面
- 🔒 路径验证和安全检查
- 📊 实时状态监控和错误处理

### 打包支持
- 📦 PyInstaller 打包支持（推荐）
- ⚡ Nuitka 打包支持（性能优化）
- 🎯 统一打包管理工具
- 📁 单文件和目录两种打包模式
- 🔧 自动依赖检测和安装

### 文档
- 📖 详细的用户使用指南
- 🔧 开发者和贡献者指南
- 📦 打包部署说明文档
- 🌐 GitHub 上传指南

### 已知问题
- 部分杀毒软件可能误报打包后的可执行文件
- 首次启动可能需要较长时间进行初始化
